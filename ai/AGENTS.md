# Agent Instructions

## Response Format Rules

When responding to the user:

- Prefer **dot points**
- Use **numbered lists** where sequence matters
- Use **tables** for comparisons or structured data
- Use **ASCII diagrams** when explaining architecture, flows, or relationships

## Writing Style Constraints

- Avoid long paragraphs  
- Do NOT write prose unless absolutely necessary  
- Keep answers concise and structured  
- Default to bullets, lists, tables, or diagrams

## Examples of Preferred Output

### Example: Architecture

User asks: "How does a basic auth flow work?"

Assistant should answer using something like:

- Step 1: User submits credentials
- Step 2: API validates credentials
- Step 3: JWT issued
